==============================
     Simple Alarm Clock
==============================

Description:
A Python program that lets the user set an alarm.
The program waits until the set time, then plays
a sound or prints a message to alert the user.

Features:
- Set alarm in HH:MM:SS format
- Waits until alarm time
- Alerts user with message or sound
- Runs in background until triggered

How to Run:
1. Save the file as 01_main.py
2. Open VS Code terminal.
3. Run the program with:
   python3 01_main.py
